Описание тестового фреймворка.

Фреймворк построен с ипользованием шаблона проектироваия PageObject. Файлы страниц содержат только компоненты страницы. Для упрощения работы со страницами добавлен промежуточный слой, в ней реализована логика взаимодействия к элементами страницы. 

В тестах произодится ининциализация необходимых "хелперов" и через них вызов методов работы со страницей.


Запуск тестов.
Запуск тестов выполняется командой mvn clean test
